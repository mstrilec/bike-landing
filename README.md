# Bike Landing
This landing page is designed for a bike company, showcasing their products and services. It has been created using HTML, CSS and JavaScript.

# Getting Started:
To view the Bike Landing Page, simply visit https://mstrilec.github.io/bike-landing/ on your web browser.

# Navigation:
The navigation bar at the top of the page allows users to easily navigate to different sections of the website. The navigation bar includes links to Home, Bikes, Accessories, and Contact Us sections.

# Sections:
The website consists of the following sections:

1. Home: The home section provides a brief introduction to the bike company and displays an image of a bike.

2. Bikes: The Bikes section displays the available bikes for purchase. Each bike has a picture, price, and a short description.

3. Accessories: The Accessories section displays the available accessories for purchase. Each accessory has a picture, price, and a short description.
